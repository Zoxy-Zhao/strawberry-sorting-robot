# 系统设计规格书

## 1. 项目定位

面向草莓等易损果实的**眼手力柔性抓取机器人**，采用"视觉识别 + 触觉反馈 + 力控执行 + 端侧 TinyML 推理"闭环架构，实现安全、自适应、低损伤的抓取与分拣。

### 核心问题

| 问题 | 说明 |
|------|------|
| 看得见但抓不准 | 只能识别位置，无法根据目标形态和抓取状态动态调整 |
| 抓得住但容易伤 | 夹持力过大压坏草莓，过小又滑脱 |
| 执行刚性强、安全性不足 | 遇障碍、姿态异常或夹爪受阻时不能快速反应 |

### 关键词

眼手力协同 · 易损果实抓取 · 柔性末端执行器 · 实时力控 · 端侧 TinyML · 神经拟态式异常保护

---

## 2. 系统总体架构

四层架构：

```
┌──────────────────────────────────────────────┐
│  视觉感知层（眼）── 树莓派                      │
│  图像采集 → 目标识别 → 坐标映射 → 指令下发       │
├──────────────────────────────────────────────┤
│  运动执行层（手）── 6DoF 机械臂 + 柔性夹爪       │
│  空间运动 → 柔性接近 → 抓取 → 提升搬运 → 放置    │
├──────────────────────────────────────────────┤
│  触觉力觉层（力）── 薄膜压力传感器               │
│  接触检测 → 力值变化 → 左右平衡 → 异常阻力       │
├──────────────────────────────────────────────┤
│  控制决策层（脑）── RA6M5                       │
│  力控闭环 → TinyML 推理 → 异常中断保护           │
└──────────────────────────────────────────────┘
```

### 模块分工

**树莓派（视觉大脑）**：图像采集、草莓识别、成熟度判断、坐标映射、任务调度

**RA6M5（控制小脑）**：运动指令解算、机械臂控制、高速压力采样、抓取状态判断、TinyML 推理、异常力中断保护

## 3. 五大核心功能

### 功能一：草莓目标视觉识别与定位

- **输入**：相机图像
- **输出**：目标坐标、类别/成熟度标签、抓取优先级
- **初级版**：OpenCV 固定背景 + 颜色/轮廓识别 + 标定坐标映射
- **进阶版**：轻量 YOLO 检测 + 成熟度区分 + 多目标排序

### 功能二：机械臂轨迹执行与柔性接近

动作流程：待机 → 接收坐标 → 安全高度接近 → 缓慢下降 → 夹爪闭合 → 接触检测触发力控

五个控制阶段：
1. 空间运动阶段（快速）
2. 接触检测阶段（减速）
3. 力控抓取阶段（精细）
4. 提升搬运阶段
5. 放置释放阶段

### 功能三：触觉反馈与接触检测

- 夹爪接触面贴柔性薄膜压力传感器
- 接触前：夹爪闭合速度正常
- 检测到接触后：立即降速，转力控模式
- 力值接近上限：停止增压，维持稳定夹持
- 异常陡升：立即触发保护

### 功能四：端侧 TinyML 抓取状态识别

- **输入**：最近 100~300ms 的压力变化曲线
- **输出**：5 类抓取状态
  - 未接触 → 继续闭合
  - 轻触未稳 → 稍增夹持力
  - 抓取稳定 → 维持当前力
  - 过压风险 → 降力/停止加压
  - 滑脱风险 → 微增力并重新稳定
- **模型**：1D-CNN 或一维卷积+全连接，量化后部署到 RA6M5

### 功能五：神经拟态式异常保护

模仿生物反射弧，RA6M5 通过硬中断快速响应异常力。

- **触发条件**：力值超阈值、力值斜率突增、执行卡滞、人为阻挡
- **响应动作**：关闭/降低 PWM → 停止危险动作 → 进入安全状态 → 上报异常

---

## 4. 创新点

1. **视觉—触觉—力觉多模态闭环**：视觉引导 + 触觉反馈 + 力觉调节的完整闭环
2. **端侧 TinyML 参与抓取决策**：压力时序实时推理，辅助夹持控制
3. **神经拟态式快速保护**：硬中断异常力响应，避免压伤和碰撞
4. **柔性夹爪设计**：硅胶/TPU 包覆接触面 + 薄膜压力传感器
5. **异构双脑协同**：树莓派高层调度 + RA6M5 底层实时控制

## 5. 硬件方案

### 主控平台

| 平台 | 职责 |
|------|------|
| RA6M5 | 实时采样、力控闭环、中断保护、TinyML 推理、舵机控制 |
| 树莓派 | 图像处理、目标识别、坐标映射、任务调度、日志可视化 |

### 执行机构

- 6DoF 桌面机械臂（CH5→CH1→CH0→CH2→CH3→CH4）
- 柔性接触夹爪（两指，指端硅胶/TPU 包覆，内嵌薄膜压力传感器）

### 传感器

- 相机模块（视觉识别）
- 柔性压力传感器（夹爪受力检测）
- 可选：限位/霍尔/编码器（姿态反馈）、IMU 或电流检测（状态增强）

## 6. 展示指标

### 设计目标

| 类别 | 指标 |
|------|------|
| 识别 | 草莓识别成功率、定位偏差 |
| 抓取 | 抓取成功率 ≥95%、滑脱率 <5%、单次耗时 |
| 力控 | 平均夹持力、峰值力控制、不同尺寸适应性 |
| 安全 | 异常响应时间 <10ms、保护触发成功率 |
| 质量 | 压伤率 <5%、表面损伤程度 |

### 对比实验设计

1. **无力控 vs 有力控**：验证触觉反馈的价值（成功率、损伤率、滑脱率）
2. **固定阈值 vs TinyML 自适应**：验证 TinyML 不是摆设（不同硬度稳定性、过压/滑脱次数）
3. **无保护 vs 有保护**：验证拟态机制有效（遇阻停止时间、持续挤压情况）

- 板卡：`DShanMCU RA6M5`
- 当前新工程：`D:\e2studio_test\Robotic_arm`

已确认引脚和配置：

- `P511 -> IIC1_SDA1`
- `P512 -> IIC1_SCL1`
- `IIC1`
- `0x40`
- `7-bit`
- `IPL 12`

## 2. PCA9685 驱动板

- 类型：16 路 I2C 舵机驱动板
- 芯片：`PCA9685`
- 与 MCU 关系：MCU 只负责 I2C 寄存器控制，PWM 由 PCA9685 输出

## 3. 接线关系

逻辑侧接线：

- `RA6M5 P511 -> PCA9685 SDA`
- `RA6M5 P512 -> PCA9685 SCL`
- `RA6M5 3V3 -> PCA9685 VCC`
- `RA6M5 GND -> PCA9685 GND`

动力侧接线：

- 外部降压模块正极 -> PCA9685 舵机电源正极
- 外部降压模块地 -> PCA9685 舵机电源地
- 外部降压模块地与 RA6M5 地共地

## 4. OE 引脚

- 当前阶段可以不接
- 如果后续出现输出无效或异常，优先尝试将 `OE` 接地
- 如果以后要做硬件级停机，可以把 `OE` 接到一个 GPIO

## 5. 测试时供电建议

- RA6M5：优先由电脑 USB 供电
- PCA9685 逻辑侧：由 RA6M5 `3V3` 供电
- 舵机：独立外部电源供电
- 烧录时逻辑线可以保持连接
- 首轮调试建议烧录完成后再上舵机大电源

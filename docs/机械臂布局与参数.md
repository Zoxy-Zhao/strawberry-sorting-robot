# 6 舵机机械臂布局与参数

## 1. 推荐关节分配

| 通道 | 规格 | 位置 | 作用 |
|---|---|---|---|
| `CH5` | 30kg / 270° | 底座旋转轴 | 整机左右转向 |
| `CH1` | 25kg / 180° | 肩关节 | 大臂抬升，主承重 |
| `CH0` | 20kg / 180° | 肘关节 | 小臂前伸/回收 |
| `CH2` | 20kg / 180° | 腕部俯仰 | 末端上仰/下压 |
| `CH3` | 20kg / 180° | 腕部旋转 | 夹爪方向微调 |
| `CH4` | 20kg / 180° | 夹爪开合 | 执行抓取 |

机械链顺序：

`CH5 -> CH1 -> CH0 -> CH2 -> CH3 -> CH4`

## 2. 推荐尺寸

| 项目 | 值 | 说明 |
|---|---:|---|
| `H1` | `95 mm` | 底座到肩轴高度 |
| `L2` | `130 mm` | 肩轴到肘轴 |
| `L3` | `140 mm` | 肘轴到腕俯仰轴 |
| `L4` | `45 mm` | 腕俯仰到腕旋转 |
| `L5` | `55 mm` | 腕旋转到夹爪根部 |
| `Lf` | `45-50 mm` | 单个夹指长度 |
| `Wg` | `60-70 mm` | 最大开口 |
| 软垫厚度 | `3-5 mm` | TPU/硅胶 |

## 3. 结构原则

- 末端尽量轻
- 腕部结构尽量短
- 大臂和小臂不要为了追求大范围而过长
- 装配时所有舵机先打到中位再装舵盘

## 4. 建议零位装配

- `CH5` 底座：机械臂朝前
- `CH1` 肩关节：大臂略抬起
- `CH0` 肘关节：小臂向内折起
- `CH2` 腕俯仰：夹爪大致朝下
- `CH3` 腕旋转：夹爪朝前
- `CH4` 夹爪：半开

## 6. 当前限制

- 当前姿态值是首版起点
- 每个关节是否反向，需要整臂装配后实测 `reversed`
- 当前 `safe_us` 仍是保守值，装配后必须二次校准

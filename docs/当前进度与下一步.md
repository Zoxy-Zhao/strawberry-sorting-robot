## 参考信息

### HOME 姿态最终参数

```
CH5 BASE         135°   底座朝向正前
CH1 SHOULDER      70°   肩关节向前倾（增大=向前）
CH0 ELBOW         70°   肘关节适度弯曲（增大=向前）
CH2 WRIST_PITCH  150°   腕部俯仰（90°=笔直，60°=向前）
CH3 WRIST_ROTATE  80°   腕部旋转居中（机械中立位为80°）
CH4 GRIPPER      110°   夹爪半开待抓（40°=夹紧，160°=全开，reversed=true）
```
